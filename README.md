# nuxt-crud pnpm Monorepo

This is a pnpm workspaces monorepo containing three packages:
- `cli`: Command-line interface tools
- `module`: Core logic or shared modules
- `docs`: Documentation site or files

## Getting Started

- Install dependencies: `pnpm install`
- Run scripts from the root or individual packages as needed.

## Structure
- `/cli` - CLI tools
- `/module` - Shared logic
- `/docs` - Documentation
